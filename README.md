## TiddlyWiki

This project includes a TiddlyWiki instance configured with:
- Port: 8090
- Theme: Default
- Plugins: markdown, toc, highlight, katex


To start the wiki server:
```bash
cd /Users/aalberici/projects/awe2
tiddlywiki wiki --listen port=8090
```
